# Direttiva: Sezioni Progetti in Light Mode (COMPLETATA)

## Obiettivo
Creare un forte stacco visivo per le sezioni dei case study utilizzando un fondo bianco.

## Estetica Sezione Light
- **Background**: `#ffffff`
- **Heading/Title**: `#030303`
- **Body Text**: `#424245` (grigio scuro Apple-style)

## Estetica Card Light
- **Background**: `rgba(0, 0, 0, 0.02)`
- **Border**: `1px solid rgba(0, 0, 0, 0.08)`
- **Shadow**: Ombre più sfumate e meno profonde per un look "airy".
- **Hover**: Incremento del contrasto del bordo.

## Strumenti/Script
- `execution/apply_light_theme.py`: Aggiornamento CSS e componenti.

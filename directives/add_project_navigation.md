# Direttiva: Navigazione Sequenziale Case Study (COMPLETATA)

## Obiettivo
Migliorare la ritenzione dell'utente permettendo la navigazione fluida tra i case study.

## Logica
- Identificare l'indice del progetto attuale nell'array `projects`.
- Calcolare l'indice del progetto successivo (con ritorno all'inizio se è l'ultimo).
- Mostrare un link "Next Project: [Titolo]" in fondo a ogni pagina.

## UI
- Link spazioso, centrato, con un'estetica editoriale.
- Transizione fluida verso la nuova pagina.

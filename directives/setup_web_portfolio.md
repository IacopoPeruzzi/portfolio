# Direttiva: Setup Web Portfolio

## Obiettivo
Inizializzare un ambiente di sviluppo web moderno per il portfolio personale, garantendo una struttura pulita e pronta per un design premium.

## Input
- Nome progetto: `web-app` (all'interno della root del portfolio)
- Tech Stack: Vite + React + Vanilla CSS (come richiesto per massima flessibilità).

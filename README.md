# Iacopo Peruzzi Portfolio

Progetto gestito seguendo l'architettura a 3 livelli.

## Struttura
- `directives/`: Standard Operating Procedures (SOP) e istruzioni di alto livello.
- `execution/`: Script Python deterministici per l'esecuzione dei task.
- `.tmp/`: File temporanei e dati di elaborazione.
- `.env`: Configurazioni e segreti (non committare).

## Stato del Progetto
- [x] Struttura directory inizializzata.
- [x] Direttiva di sistema configurata.
- [ ] Definizione obiettivi portfolio.
